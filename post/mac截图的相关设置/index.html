Mac截图的相关设置

Jul 1, 2019
1 min read

选区域截图默认快捷键

Shift+command+4

默认保存目录

~/Desktop

修改目录

defaults write com.apple.screencapture location "~/Pictures/ScreenShot"

截图默认后缀

png

修改后缀

defaults write com.apple.screencapture type jpg

删除默认时间后缀

defaults write com.apple.screencapture "include-date" 0

修改默认的文件名称

defaults write com.apple.screencapture name "screenshot"

参看配置

defaults read com.apple.screencapture
输出如下：
{
"last-messagetrace-stamp" = "583174121.285828";
"last-selection" =     {
    Height = 529;
    Width = 1041;
    X = 507;
    Y = 262;

"last-selection-display" = 0;
location = "~/Pictures/ScreenShot";
name = screenshot;
type = jpg;
}

缺点分析

mac自带的截图工具，不能修改时间的格式，而且这种格式有空格，不利于作为文件名。虽然可以去除时间后缀，但是时间信息很重要。同时，即使没有时间后缀，生成的图片名中还是存在空格。鉴于上面复杂的问题，我权衡利弊之后果断放弃这个mac自带的截图工具。

更好的截图工具

腾讯开发的一款名为”截图”的截图工具，它可以自定义目录，自动生成的文件名包含时间信息且没有空格。
